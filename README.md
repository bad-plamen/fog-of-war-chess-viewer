# fog-of-war-chess-viewer
Визуализатор за Fog of War Chess - показва дъската от FEN стринг, получен от WebSocket съобщенията на играта. Поддържа двама играчи (бели и черни) с пълна статистика на фигурите.

Fog of War Chess Board Viewer - displays the chessboard from FEN strings received via WebSocket messages. Supports two players (white and black) with full piece statistics.
